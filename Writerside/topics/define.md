# Defined values

## gametype

|gametype|Description|
|:---|:---|
|slot|Slot games|
|table|Table games, will be available in future. (Schedule TBD)|

## gameid

"gameid" is a unique ID for each games, please refer to the provided SLOT_SPEC document

You will be informed when a new game is added via "SLOT_SPEC" document.

## currency

Supported currencies are identified with 3-digit ISO 4217 codes.
please refer to the provided SLOT_SPEC document "Currency support" sheet.

You will be informed when a new currency is added via "SLOT_SPEC" document.

## language
