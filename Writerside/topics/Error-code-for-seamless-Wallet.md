# Error code for seamless Wallet

Below table contains the error codes that should be returned by Casino Operator in the response to Seamless Wallet API calls.


| Name |Data Type| Description                                                                                                                                                                                                                                                                                                        | Remark   |
|:-----|:---:|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------|
| 0    |string| Success                                                                                                                                                                                                                                                                                                            | Required |
| 1    |string| Insufficient balance. The error should be returned in the response on the Bet request                                                                                                                                                                                                                              | Required |
| 2    |string| Player not found or is logged out. Should be returned in the response on any request sent by Lucky Monaco if the player can’t be found or is logged out at Casino Operator’s side.                                                                                                                                 | Required |
| 3    |string| Bet is not allowed. Should be returned in any case when the player is not allowed to play a specific game. For example, because of special bonus.                                                                                                                                                                  | Required |
| 4    |string| Player authentication failed due to invalid, not found or expired token.                                                                                                                                                                                                                                           | Required |
| 5    |string| Invalid hash code. Should be returned in the response on any request sent by Lucky Monaco if the hash code validation is failed.                                                                                                                                                                                   | Required |
| 6    |string| Player is frozen. Casino Operator will return this error in the response of any request if player account if banned or frozen.                                                                                                                                                                                     | Required |
| 7    |string| Bad parameters in the request, please check post parameters.                                                                                                                                                                                                                                                       | Required |
| 8    |string| Game is not found or disabled. This error should be returned on Bet request if the game cannot be played by some reason. Bet result request with winning amount should be processed as intended, even if the game is disabled.                                                                                     | Required |
| 50   |string| Bet limit has been reached. The code is relevant for regulated markets.                                                                                                                                                                                                                                            | Required |
| 100  |string| Internal server error. Casino Operator will return this error code if their system has internal problem and cannot process the request at the moment and Operator logic requires a retry of the request. <br/>Request will follow Reconciliation process                                                           | Required |
| 120  |string| Internal server error. Casino Operator will return this error code if their system has internal problem and cannot process the request and Operator logic does not require a retry of the request. <br/> Request will NOT follow Reconciliation process                                                            | Required |
| 130  |string| Internal server error on EndRound processing. Casino Operator will return this error code if their system has internal problem and cannot process the EndRound request, and Operator logic requires a retry of the request. <br/>This error code should be used for Endround method only and not for other methods | Required |
| 210  |string| Reality check warning.                                                                                                                                                                                                                                                                                             | Required |
| 310  |string| Player’s bet is out of his bet limits. Should be returned if player’s limits have been changed, and the bet is out of new limit levels. Game client will show a proper error message, and ask player to reopen the game. After game reopening new bet limits will be applied. The error is relevant for operators that send player’s bet limits in response on Authenticate request.                                                                                                                                                                                                                                                                    | Required |
