# Freeround Integration

## Overview

Free rounds for promotions can be provided to users.

!! Transfer server method is not currently supported.

## Usage

If you request luckymonaco, you can provide free rounds to specific users.

## APIs

1. get_info - Get freeround information.
2. bet_info - Obtain a list of betting amounts for specific slots and currencies.
3. create - Create a freeround.
4. delete - Remove freeround.
