Kyren V1.00 Help

Session Expired

Overview

Using this method the Kyren system will notify Casino Operator that player’s session has expired in Kyren system due to long inactivity or game closing.

The method is optional and is not sent to the Casino Operator by default. In case Casino Operator needs this method to be sent, they should ask the Kyren’s team member when performing the integration.

Request

Requested "Session Expired" API URL will be notified individually, for security reasons.
