LuckyMonaco V4.02 Help

Jackpot Winners

Overview

The Jackpot Winners feed provides information about jackpot winnings including progressive jackpot wins and community share part. An operator will get in the response a list of the winners in JSON format. Maximal period of Jackpot Winners DataFeeds for which data can be transferred is not more than 30 days

Request

Requested "Jackpot Winners" API URL will be notified individually, for security reasons.
