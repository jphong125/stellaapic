LuckyMonaco V4 Help

Session Expired

Overview

Using this method the Lucky Monaco system will notify Casino Operator that player’s session has expired in Lucky Monaco system due to long inactivity or game closing.

The method is optional and is not sent to the Casino Operator by default. In case Casino Operator needs this method to be sent, they should ask the Lucky Monaco’s Technical Support for additional configuration

API URL

Requested game session API URL will be notified individually, for security reasons.

Example of failure response from LuckyMonaco API servers.
