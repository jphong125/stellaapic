LuckyMonaco V4.02 Help

Jackpot Win (Reserved for future development)

Overview

Using this method a Lucky Monaco system will notify Casino Operator about Jackpot winning.
Notification is asynchronous and may come to the operator with a short delay after game round is over.
Operator should handle the transaction in their system and send the jackpot win transaction id back to the Lucky Monaco.

Important : The call is idempotent, i.e. sending result again with the same reference number creates only one transaction.

Request

Requested "Jackpot win" API URL will be notified individually, for security reasons.
