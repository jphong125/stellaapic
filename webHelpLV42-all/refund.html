LuckyMonaco V4.02 Help

Refund

Overview

Lucky Monaco system may use this method to rollback a round on the Casino Operator side, in order to reverse the transaction and adjust player’s balance.
When receive a refund request Operator have to return money back to player’s balance.

  • Important : The call is idempotent, i.e. sending refund for existing bet again only creates one transaction.

  • Important : If bet transaction is not found then nothing should happen on the Casino Operator side and success (0) or specific error code for this situation should be returned.

  • Important : It is automatically executed when the operator executes "cancel round".

Request

Requested "Refund" win API URL will be notified individually, for security reasons.
