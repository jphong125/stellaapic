LuckyMonaco V4.02 Help

Session Expired (Reserved for future development)

Overview

Using this method the Lucky Monaco system will notify Casino Operator that player’s session has expired in Lucky Monaco system due to long inactivity or game closing.

The method is optional and is not sent to the Casino Operator by default.
In case Casino Operator needs this method to be sent, they should ask the Lucky Monaco’s team member when performing the integration.

Request by lucky monaco

Requested "Session Expired" API URL will be notified individually, for security reasons.
