LuckyMonaco V4 Help

Get Casino Games

Overview

Using this method Casino Operator will get the list of casino games available for the integration.

API URL

Requested get casino games API URL will be notified individually, for security reasons.

Request parameters

Name

Data Type

Description

Remark

secureLogin

string

User name for authentication in the Casino Game API service

Required

token

string

Token of the player from Authenticate response

Required

Examples

METHOD

POST

URL

https://<API URL>/V4/GetCasinoGame

HEADER

Content-Type: application/json

Example of HTTP BODY

{ "secureLogin" : "<partnerId>", "token" : "<token>" }
