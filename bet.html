Stella V1.00 Help

Bet

Overview

Using this method Stella system will check the player balance on Casino Operator side to ensure they still have the funds to cover the bet.
Amount of the bet must be subtracted from player balance in Casino Operator system.
Players connection was lost in any cases so Stella will do retransmission 3 times, then do not receive any bet response. Stella will request a cancel bet.

Important: The call is idempotent, i.e. sending bet again only creates one transaction.

Requested "Bet" API URL will be notified individually, for security reasons.
